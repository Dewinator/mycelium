affective state (ebene 1b)

vier persistente regelvariablen. werden automatisch von remember, recall, absorb, digest aktualisiert. frustration hat zeit-decay (half-life aus agent_affect.decay_half_life_hours).

recall bias (ableitung)

wie der aktuelle state dein recall gerade beeinflusst. Δk = zusätzliche memory-treffer, threshold = score-cutoff (null = caller-default), spread_wide = aggressive spreading activation, adjacent_tags = angrenzende tags einbeziehen.

belief sidecar (ebene 3 — active inference)

pymdp-agent auf 127.0.0.1:18790 als launchagent ai.openclaw.belief. wird von infer_action angefragt; empfiehlt recall / research / ask_teacher per minimierter expected free energy.

skill performance (was hat funktioniert?)

success-rate pro skill × task-type, gefüttert aus digest-aufrufen mit tools_used. grundlage für prime_context und recommend_skill.

kausale annotationen (ursache → wirkung)

zwischen erfahrungen gezogene kausal-kanten. auto-ingested mit source=digest_extracted, bestätigt mit source=explicit/user_confirmed. keine pearl-kausalität — annotations-schicht. nutze causal_chain um einer kette zu folgen.

motivation sidecar (ebene 4)

stimulus-sammler auf 127.0.0.1:18792 als launchagent ai.openclaw.motivation. schleife: collect → score → generate → drift, alle 60 min. quellen: HackerNews, RSS, git activity, GitHub trending. relevanz-score = 0.20·memory + 0.35·affect + 0.25·soul + 0.20·intention.

kopplung an die neurochemie (migration 043)

drei rückkanäle: (a) nach jedem gescorten stimulus feedback an dopamin/noradrenalin je band, (b) band-thresholds shiften dynamisch mit NE (hoch = enger, niedrig = breiter), (c) drift-rate × (2−5-HT): niedriges serotonin lässt dormant tasks schneller „unruhig" werden.

agent-genome (ebene 5c)

persistente identitäten mit values, interests, verhaltens-parametern. fitness = 0.40·avg_outcome + 0.25·growth + 0.20·breadth + 0.15·autonomy. reproduktion ist approval-gated — keine automatische züchtung.

nightly consolidation (biomimetisch)

LaunchAgent ai.openclaw.sleep läuft täglich 03:00. Biologisches Vorbild: SWS (forget_weak → consolidate → dedup) · REM (cluster → dedup_lessons → promote_trait) · Metacognition/DMN (update_self_model) · Weekly snapshot_fitness (Sonntag). Log: ~/.openclaw/sleep/logs/
